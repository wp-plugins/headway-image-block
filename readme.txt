A simple Headway Themes block for adding images to your website.

== Description ==

The "Headway Image Block" is an add-on block for Headway Themes (http://headwaythemes.com) that lets you easily add a single image anywhere on your website.

== Installation ==

1. Upload the `headway-image-block` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Open the Headway Visual Editor in Grid Mode
1. Create a new block and choose "Image" to add the block to your layouts.

== Frequently Asked Questions ==

= Do you have to have Headway Themes installed to use this plugin? =

Yes. This plugin is built specifically for Headway Themes.

= How many images can you add to a single image block? =

Just one. The image block is basically for simplifying the ability to add images to Headway. For slideshows or galleries, use one of the other blocks.
